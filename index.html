Sticker Journey Map

Open Source

借助 Nano Banana 模型，将日常照片转化为专属 Q 版贴纸。它能自动读取图像的 GPS 坐标并在地图上点亮足迹，让静态的旅行回忆在地理空间中生动再现。 Utilizing the Nano Banana model, it transforms everyday photos into custom chibi stickers. By auto-extracting GPS metadata to pin footprints on a map, it brings static travel memories to life in geographic space.

构建洞察Builder's Insight 一个由个人兴趣驱动的开源实践。技术层面上，将端侧模型生成与前端地图交互融合；产品层面上，则是将"电子手账"这一感性诉求赋予了可编程的代码躯壳。 An open-source practice driven by personal passion. Technically, it blends edge-side model generation with front-end map interactions; functionally, it gives the emotional desire for a "digital scrapbook" a programmable code shell.

Xmindflow

Open SourceLangGraph

把 Xmind 一步到位输出原型图和 PRD 的 multi-agent 工作流。 Multi-agent workflow that outputs prototypes and PRD directly from Xmind in one step.

策展人笔记Curator's Note 让合适的模型做合适的事。通过编排协作网络，复杂逻辑交由 GPT-4o 解析，高保真代码则由 Claude 3.5 呈现。精准的上下文流转让交付过程无需人工干预，这是对 LLM 能力边界的一次探索与实践。 Letting the right model do the right job. By orchestrating a multi-agent network, complex logic is decoded by GPT-4o, while high-fidelity code is crafted by Claude 3.5. Precise context passing enables zero-intervention delivery, serving as an exploration into LLM boundaries.
